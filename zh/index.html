Macaca DataHub

支持研发全周期的数据环境解决方案

快速上手

多环节覆盖

支持从本地开发阶段，到集成测试阶段，以及上线前验证阶段的一系列数据环境需求。

去中心化

数据托管采用去中心化设计，本地研发阶段每项实例都拥有一份独立的完整数据，数据文件可跟随 Git 版本化管理。

数据流动管理

采用单向数据流动的原则管理你的研发数据流，使当前项目下的数据内容及时变更。

场景管理

支持多场景，能够根据场景名称进行数据分组，随时还原所需场景数据，同时提供了场景数据的增、删、改，可以通过 DataHub 的面板界面进行操作。

文档一致性

将 Mock 数据与 Schema 字段描述整合处理，自动生成接口文档，使得文档能够与交互字段随时保持一致。

灵活录入

支持多种导入、录入方式，兼备代理功能，支持将最近请求的实时响应保存下来，便于归档。也就是说你可以通过已归档的快照随时复现当时的场景。

开发体验

DataHub 提供 UI 操作板，将大大提升你的研发体验。
